üîß Usage

Automatic Reviews: Configure the GitHub webhook to point to /webhook. Reviews will automatically trigger on PR open and new commits.

Manual Reviews: Comment @smaug123-robocop review on any PR to trigger a review.

Cancel Reviews: Comment @smaug123-robocop cancel on any PR to cancel pending reviews.

üîï Review Suppression

You can control whether robocop automatically reviews your pull requests:

Disable Reviews via PR Description

Include @smaug123-robocop disable-reviews anywhere in your PR description to suppress automatic reviews.

Commands

  • @smaug123-robocop disable-reviews - Disable automatic reviews and cancel pending reviews
  • @smaug123-robocop enable-reviews - Enable automatic reviews and trigger review for current commit
  • @smaug123-robocop review - Request a one-time review even if reviews are disabled

State Persistence: Once disabled, reviews stay disabled until explicitly enabled.
